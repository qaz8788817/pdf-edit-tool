# pdf-edit-tool
An easy tool for edit pdf files.

Using Python package.  
`pip install customtkinter pypdf`

Generate .exe program: `pyinstaller --noconsole --onefile pdf_edit.py`  
If you want a cute .ico showing on your Desktop:
1. Get the cute icon (ex: test.ico), and put the icon file and python file in the same folder.
2. Run `pyinstaller --noconsole --onefile --icon=test.ico pdf_edit.py`
3. You got the cutest pdf tool in the build dir. You can use it right now.

More function about the tool, check this website: https://qaz8788817.github.io/projects/5-pdf-edit/
